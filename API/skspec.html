API¶

Primary Structures: Spectrum, Spectra, SpecStack¶

Each DataStructure below is the Spectroscopic extension of a Pandas object:

  • Spectrum : pandas.Series
  • Spectra : pandas.DataFrame
  • SpecStack : pandas.Panel

As such, each is constructed similar to its pandas equivalent:

Spectra(array, index, columns, specunit='nm', varunit='s')

Please see the basic tutorial for more on scikit-spectra datastructures and their instantiations.

Spectrum¶

spectra.Spectrum(*args, **kwargs) Compliment to pandas Series: single array of spectral values with spectral Index.

Spectra¶

anyspectra.AnyFrame(*dfargs, **dfkwargs) Spectra with default non-strict columns or index
spectra.Spectra(*dfargs, **dfkwargs) Provides core Spectra composite pandas DataFrame to represent a set of spectral data.
timespectra.TimeSpectra(*dfargs, **dfkwargs) Spectra that enforces a TimeIndex.
tempspectra.TempSpectra(*dfargs, **dfkwargs) Spectral Index (rows) with Temperature Variation (columns)

SpecStack¶

specstack.SpecStack(data[, keys, name, ...]) Stack for just storing timespectra objects.
