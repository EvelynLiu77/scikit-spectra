pandasplotdata Module¶

class skspec.chaco_interface.pandasplotdata.PandasPlotData(dataframe, columnlabel='columns', indexlabel='index')[source]¶

Bases: chaco.abstract_plot_data.AbstractPlotData

Chaco requires a PlotData interface to manage plot/data mapping; however, pandas already is its own nice data handler, so this class mostly provides a wrapper that tries to be a hidden middle man. As such, I try to make this PandasPlotData feel like a pandas dataframe. It would be nice to have chaco communicate with a dataframe directly, but for now it must do so through this liason class.
