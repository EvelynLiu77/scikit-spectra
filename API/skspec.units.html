class skspec.units.specunits.SpecUnit(**kwargs)[source]¶

Bases: skspec.units.abcunits.ConversionUnit

Spectroscopy units (nm, m, ev etc...) Allow for recpricol units such as cm-1; used by converters. All conversions go through unit, considered the core unit. Adds category to distinguish between wavelength, wavenumber, energy, frequency.

category = ''¶
symbol¶

Used by plotting, especially correlation analysis. For example,

label1 = r’$ar{A}(%s_1)$’ % symbol

Where symbol is this. Don’t forget to wrap in r’$...$’ for tex.
