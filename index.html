Welcome to scikit-spectra¶

scikit-spectra is a set of wrappers and tools written using the pandas library with the intention of exploring spectral data, especially UV-Vis spectroscopy.

What’s New¶

I finally created this site which is generously hosted by GitHub. The source code is available for download; and the PyPi package is available http://pypi.python.org/pypi/scikit-spectra/ installation.
