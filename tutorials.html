    Two-Dimensional Correlation Spectroscopy¶

    Two-Dimensional Correlation Spectroscopy (2DCS) is a field spectral analysis that pertains to analyzing the the relationship between the instantaneous changes in a spectrum. It is useful for resolving information about the order and time at which peaks grow, fade or shift. Scikit-spectra tries to provide an API for 2DCS. For literature on the subject, see Noda/Ozaki.
